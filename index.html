Politique de confidentialité

Dernière mise à jour : 7 avril 2026

En résumé : Vos données restent sur votre téléphone. Nous ne vendons, ne partageons et ne monétisons aucune donnée personnelle.

1. Qui sommes-nous

Résumé d'Appel (ci-après « l'Application ») est une application mobile de transcription et de résumé automatique d'appels téléphoniques, développée par un développeur indépendant.

2. Données collectées

L'Application peut collecter et traiter les données suivantes :

  • Informations de compte : nom, adresse email, mot de passe (chiffré)
  • Contacts : nom, numéro de téléphone, email, entreprise (uniquement les contacts que vous ajoutez manuellement ou importez)
  • Enregistrements audio : les appels que vous choisissez d'enregistrer
  • Transcriptions et résumés : générés par le service d'IA que vous configurez
  • Métadonnées d'appels : date, durée, direction (entrant/sortant)

3. Stockage des données

Toutes vos données sont stockées localement sur votre appareil (base de données SQLite). La sauvegarde vers un serveur est optionnelle et sous votre contrôle.
  • Stockage local : base de données SQLite sur votre téléphone
  • Sauvegarde serveur : si activée, vos données sont envoyées vers votre serveur personnel de manière chiffrée (HTTPS)
  • Clé API : stockée de manière sécurisée via le trousseau de votre appareil (Secure Store)

4. Services tiers

L'Application utilise un service d'intelligence artificielle (au choix de l'utilisateur) pour générer les transcriptions et résumés :

  • Anthropic (Claude)
  • OpenAI (ChatGPT)
  • Google (Gemini)
  • Microsoft (Copilot)

L'audio est envoyé au service choisi uniquement pour la transcription. Nous vous invitons à consulter la politique de confidentialité du service que vous choisissez. Votre clé API personnelle est utilisée ; nous n'avons pas accès à vos requêtes.

5. Permissions requises

  • Microphone : pour enregistrer les appels
  • État des appels : pour détecter automatiquement les appels entrants et sortants
  • Contacts : uniquement si vous choisissez d'importer vos contacts

6. Partage de données

Nous ne partageons, vendons ou louons aucune donnée personnelle à des tiers. Vos données ne sont jamais utilisées à des fins publicitaires.

7. Sécurité

  • Mots de passe chiffrés (bcrypt)
  • Authentification par jeton JWT
  • Clés API stockées dans le trousseau sécurisé de l'appareil
  • Communications serveur en HTTPS

8. Vos droits

Conformément au RGPD, vous pouvez à tout moment :

  • Accéder à toutes vos données (elles sont sur votre téléphone)
  • Supprimer vos données en désinstallant l'application
  • Exporter vos données via la fonction de sauvegarde
  • Vous déconnecter et supprimer votre compte serveur

9. Mineurs

L'Application n'est pas destinée aux personnes de moins de 16 ans.

10. Modifications

Nous nous réservons le droit de modifier cette politique. Toute modification sera indiquée par une mise à jour de la date en haut de cette page.
