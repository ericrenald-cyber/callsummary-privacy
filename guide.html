Guide de démarrage

Suivez ces étapes pour commencer à utiliser Résumé d'Appel en 5 minutes.

1

Téléchargez l'application

Installez Résumé d'Appel depuis le store de votre téléphone :

Google Play App Store
2

Créez votre compte

Ouvrez l'app et suivez l'onboarding :

  1. Choisissez votre langue (Français ou Anglais)
  2. Parcourez les écrans d'introduction
  3. Créez votre compte avec votre nom, email et mot de passe
C'est gratuit ! Commencez avec 3 appels/jour. Passez à Pro (9,99 $/mois) quand vous êtes prêt.
3

Choisissez votre assistant IA

Allez dans Réglages → Mon assistant IA et choisissez parmi :

  • Claude (Anthropic) — excellent pour le français
  • ChatGPT (OpenAI) — le plus connu
  • Gemini (Google) — rapide et gratuit
  • Copilot (Microsoft) — intégré à l'écosystème Microsoft
Recommandation : Pour le français, Claude et ChatGPT donnent les meilleurs résumés. Gemini est gratuit pour commencer.
4

Obtenez votre clé API (gratuit)

La clé API permet à l'app de communiquer avec l'IA pour générer vos résumés. Voici comment l'obtenir selon votre IA choisie :

✨ Claude (Anthropic)

  1. Allez sur console.anthropic.com
  2. Créez un compte (gratuit) ou connectez-vous
  3. Cliquez sur « Create Key »
  4. Donnez un nom (ex: « Resume d'Appel »)
  5. Copiez la clé qui commence par sk-ant-...

Crédits gratuits : 5$ offerts à l'inscription. Un résumé coûte environ 0,01$.

💬 ChatGPT (OpenAI)

  1. Allez sur platform.openai.com
  2. Créez un compte (gratuit) ou connectez-vous
  3. Cliquez sur « Create new secret key »
  4. Donnez un nom (ex: « Resume d'Appel »)
  5. Copiez la clé qui commence par sk-...

Crédits gratuits : 5$ offerts. Rechargez à partir de 5$ ensuite.

💎 Gemini (Google)

  1. Allez sur aistudio.google.com
  2. Connectez-vous avec votre compte Google
  3. Cliquez sur « Create API key »
  4. Copiez la clé

100% gratuit pour un usage personnel. Aucune carte de crédit requise.

💻 Copilot (Microsoft)

  1. Allez sur github.com/settings/copilot
  2. Connectez-vous à votre compte GitHub
  3. Activez Copilot et copiez votre token

Essai gratuit : 30 jours. Ensuite 10$/mois.

5

Collez votre clé dans l'app

  1. Dans l'app, allez dans Réglages → Ma clé secrète
  2. Collez votre clé API dans le champ
  3. Cliquez « Enregistrer ma clé »
Sécurité : Votre clé est stockée dans le trousseau sécurisé de votre téléphone. Elle n'est jamais envoyée à nos serveurs.
6

Enregistrez votre premier appel !

  1. Sur l'écran d'accueil, appuyez sur le gros bouton « Enregistrer un appel »
  2. Passez votre appel en haut-parleur (pour que le micro capture la conversation)
  3. Quand l'appel est terminé, appuyez sur « Arrêter »
  4. L'IA génère automatiquement le résumé en quelques secondes
Version gratuite : 3 enregistrements par jour. Passez à Pro (9,99 $/mois) pour un accès illimité.
7

Ajoutez vos contacts

  1. Allez dans l'onglet Contacts
  2. Cliquez « Importer depuis l'appareil » pour importer vos contacts
  3. Marquez vos contacts professionnels comme « Pro »
  4. Les appels de vos contacts Pro seront gérés automatiquement

💡 Astuces

Rapport quotidien (Pro) : Activez le rapport email dans Réglages → Rapport par email. Vous recevrez chaque jour un résumé de tous vos appels.
Favoris : Appuyez sur l'étoile ⭐ à côté d'un appel dans l'Historique pour le marquer comme favori et le retrouver facilement.
Recherche : Utilisez la barre de recherche dans l'Historique pour retrouver un appel par mot-clé (nom du contact, sujet discuté, etc.).
Partage : Ouvrez un appel dans l'Historique et appuyez sur « Partager le résumé » pour l'envoyer par email, SMS ou WhatsApp.
